# Overview

Steel Section Designer is a desktop application developed for structural engineers to perform steel member capacity checks and optimize steel profiles according to **AISC 360-22**.

The application enables engineers to analyze section capacities, evaluate design ratios, compare available profiles, and identify the lightest section satisfying specified loading conditions.

Designed with a modern desktop interface, it combines engineering calculations, graphical visualization, and optimization tools into a single workflow.
